####好處:  
#####1.再現個人學習過程   
**範例**   
運用R的`rnorm()`建立100個常態分配隨機變數的方法:
```
x <- rnorm(100)
```

繪圖檢查隨機變數是否呈現常態分配   
```
hist(x)
```
